# Zerodha Clone

A full-stack clone of the Zerodha trading platform built using the MERN stack.

## Current Progress
- Responsive landing page inspired by Zerodha
- Separate dashboard UI
- Backend built with Node.js and Express.js
- MongoDB database connected
- Axios integration between frontend and backend
- REST API structure implemented

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Axios
- HTML, CSS, JavaScript

## Upcoming Features
- User Authentication (JWT & bcrypt)
- Protected Routes
- User Registration & Login
- Portfolio Management
- Deployment

> 🚧 This project is currently under active development. Authentication and full backend integration are in progress.
